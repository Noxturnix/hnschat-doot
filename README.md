# hnschat-doot

An [HNSChat](https://hnschat/) bot

# Setup

Set `SESSION_ID` and `DOMAIN_ID` in environment variables. Or just copy [.env.example](.env.example) to .env and edit the file

# Compile TypeScript

```
yarn build
# or
yarn build:clean
```

# Run

```
yarn start
```
